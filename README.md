# 🐱‍👤 Pokédex Arena - Aplicación Pokémon

Aplicación web tipo Pokédex con exploración de Pokémon, captura (localStorage), modo batalla simulada, y diseño responsivo con modo oscuro.

## ✨ Características

- **Lista de Pokémon**: 151 Pokémon iniciales con imágenes, tipos y búsqueda/filtro
- **Detalle completo**: Stats con barras de progreso, habilidades, biografía
- **Sistema de captura**: Guarda tu colección en localStorage
- **Modo batalla**: Enfrenta tu Pokémon contra rivales aleatorios
- **Mi Pokédex**: Visualiza y administra tus capturas
- **UI Moderna**: TailwindCSS, animaciones, skeleton loading, lazy loading
- **Modo oscuro/ claro** con persistencia
- **SPA con hash routing** (sin frameworks)

## 🚀 Despliegue rápido con Nginx
### 📋 Requisitos previos

- Servidor Linux (Ubuntu/Debian recomendado)
- Acceso sudo
- Conexión a internet (para instalar Nginx)

### 🔧 Instalación automática (recomendada)

1. **Clona el repositorio en tu servidor:**
git clone https://github.com/asimonutti33/DevOps_PokeAle.git

cd pokeale-deploy

3. Ejecuta el Script de Despliegue:
chmod +x deploy.sh
sudo ./deploy.sh
